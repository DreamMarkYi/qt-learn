# Live2D 动作播放修改指南

---

## 1. Live2DModel.h — 添加动作管理成员

**添加头文件引用：**

```cpp
#include <Motion/CubismMotion.hpp>
#include <Motion/CubismMotionManager.hpp>
```

**在类的 private 区添加成员：**

```cpp
Live2D::Cubism::Framework::CubismMotionManager* _motionManager = nullptr;
```

**在 public 区添加方法声明：**

```cpp
// 加载并开始播放一个 motion3.json 文件，loop=true 表示循环播放
void startMotion(const QString& motionPath, bool loop = true);

// 用 deltaTime（单位：秒）推进动作，并调用 Update() 重算顶点
// 返回 true 表示本帧有参数变化
bool updateMotion(float deltaTime);
```

---

## 2. Live2DModel.cpp — 实现动作加载与更新

**在 `load()` 函数末尾，`return true` 之前，初始化 motionManager：**

```cpp
_motionManager = new CubismMotionManager();
```

**实现 `startMotion()`：**

```cpp
void Live2DModel::startMotion(const QString& motionPath, bool loop)
{
    QByteArray bytes = readFile(motionPath);
    if (bytes.isEmpty()) return;

    CubismMotion* motion = static_cast<CubismMotion*>(
        CubismMotion::Create(
            reinterpret_cast<const csmByte*>(bytes.constData()),
            static_cast<csmSizeInt>(bytes.size())
        )
    );

    motion->SetLoop(loop);
    // 优先级 3 = 强制打断当前动作
    _motionManager->StartMotionPriority(motion, false, 3);
}
```

**实现 `updateMotion()`：**

```cpp
bool Live2DModel::updateMotion(float deltaTime)
{
    _model->LoadParameters();   // 读取上一帧保存的参数（口型/眨眼等叠加用）

    bool updated = _motionManager->UpdateMotion(_model, deltaTime);

    _model->SaveParameters();   // 保存本帧参数供下次叠加
    _model->Update();           // 根据新参数重算所有 drawable 的顶点
    return updated;
}
```

**在析构函数 `~Live2DModel()` 里释放 motionManager：**

```cpp
if (_motionManager) {
    delete _motionManager;
    _motionManager = nullptr;
}
```

## 4. Live2DWidget.cpp — 启动定时器，每帧更新

**在 `initializeGL()` 末尾，`_ready = true` 之后：**

```cpp
// 加载一个动作（路径换成你自己的 motion3.json）
_model.startMotion(_dir + "/motions/idle.motion3.json", true);

// 启动 60fps 定时器
connect(&_timer, &QTimer::timeout, this, [this]() {
    float dt = _clock.elapsed() / 1000.0f;  // 毫秒转秒
    _clock.restart();
    _model.updateMotion(dt);                 // 推进动作 + Update()
    update();                                // 触发 paintGL
});
_clock.start();
_timer.start(16);   // 约 60fps
```

## 5. Live2DRenderer.cpp — 每帧重传顶点到 GPU

**在 `draw()` 的 for 循环里，`glBindVertexArray(d.vao)` 之前，添加顶点重传：**

```cpp
// 动作改变了顶点位置，需要每帧把新坐标传给 GPU
const csmInt32 vtxCount = m->GetDrawableVertexCount(di);
const Core::csmVector2* positions = m->GetDrawableVertexPositions(di);

glBindBuffer(GL_ARRAY_BUFFER, d.vboPos);
glBufferSubData(GL_ARRAY_BUFFER, 0,
    sizeof(Core::csmVector2) * vtxCount, positions);
glBindBuffer(GL_ARRAY_BUFFER, 0);

glBindVertexArray(d.vao);
glDrawElements(GL_TRIANGLES, d.indexCount, GL_UNSIGNED_SHORT, nullptr);
```
